hexo 使用过程中遇到的麻烦

没反应

命令行键入 hexo d 毫无反应

  1. 原因: 缺少包 hexo-deployer-git

hexo 从版本 2.x.x 升级到 3.x.x 删除了内置的 git 包
需要单独安装 hexo-deployer-git 包

  1. 原因: 配置文件 url 以及 deploy 未配置
